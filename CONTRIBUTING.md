# Contributing

Thank you for considering a contribution to Artificial Intelligence Cyber Shield.

## Contribution goals

Contributions should improve the repository as a defensive, educational, and research-oriented AI security knowledge base.

Useful contributions include:

1. AI security frameworks.
2. AI red teaming methodologies.
3. AI threat modeling methods.
4. AI governance and assurance references.
5. Defensive control mappings.
6. APA 7 references.
7. Diagrams and structured tables.
8. Corrections to inaccurate or outdated material.

## Quality expectations

Contributions should be:

1. Clearly written.
2. Defensive in purpose.
3. Properly sourced.
4. Structured in Markdown.
5. Consistent with APA 7 citation style where references are used.
6. Respectful of copyright and licensing constraints.

## Pull request checklist

Before submitting a pull request, check that:

1. The contribution has a clear purpose.
2. Sources are reliable and cited.
3. Claims are separated from interpretation.
4. Tables are readable in GitHub Markdown.
5. Links are working.
6. No sensitive or private information is included.
